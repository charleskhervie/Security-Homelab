### About Me

I am an incoming 3rd-year Computer Science student and DOST Scholar at the University of the Philippines Tacloban College, focusing on System Administration and Cybersecurity. 

My current work involves building projects centered around Windows Server infrastructure, Active Directory deployment, Linux hardening, script automation using PowerShell and Bash, and centralized log monitoring (SIEM).

### Technical Focus

* **Identity & Access Management:** Windows Server, Active Directory Domain Services (AD DS), Group Policy Objects (GPO), and automated object provisioning.
* **Automation & Scripting:** Infrastructure management and configuration tasks using PowerShell and Bash workflows.
* **Defensive Security:** Linux server hardening and security telemetry tracking using local SIEM environments.

### Active Projects

* **Enterprise AD Sandbox:** Programmatic deployment of organizational units, automated user provisioning loops, and password compliance enforcement.
* **Secure Web Nodes:** Hardening dual-homed Linux network nodes running Nginx to serve as isolated local web targets.
